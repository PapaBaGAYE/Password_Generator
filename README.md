# Password Generator

**generatePassword()**

Cette fonction permet de générer des mots de passe aléatoires.

    Caractères composants le mot de généré : 
            - Majuscules,
            - minicules,
            - chiffres,
            - symboles.

    Exemple :
            print(generatePassword())
            output : _i{g}/W5vSQs

Module utilisé : **random**
